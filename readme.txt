Telegram-бот по поиску отелей

База отелей находится в открытом API Hotels, который расположен на сайте rapidapi.com

Имя бота в telegram: @hotels_dip_bot

Запуск бота осуществляется из файла bot.py

Директория _app содержит файлы:
1. config.py - общие данные, в том числе токен бота.
2. function.py - все функции, необходимые для обработки данных в bot.py
3. req.py - все функции запросов к API
